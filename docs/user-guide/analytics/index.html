Analytics

Built-in dashboard for volume, ratings, and team scoreboard — with date filtering and CSV export.

What’s on the dashboard

The Analytics page lives under WooCommerce → Analytics → Order Updates. It’s a single-page dashboard with three sections:

  • Volume — total updates created, solved, deleted, and currently open. Side-by-side with the previous period for context.
  • Ratings — average rating, distribution by stars, count of comments left. NPS-style breakdown into promoters / passives / detractors.
  • Team scoreboard — per-assignee breakdown of updates handled and average rating earned, so you can see who’s doing the heavy lifting and where coaching might help.

Date filtering

Top of the page has a date-range picker with quick presets (Last 7 days, Last 30 days, Last 90 days, Year to date, custom range). Every section recomputes when the range changes.

How fresh is the data?

Analytics results are cached — the first visit on a given date range runs the queries; later visits inside the cache window read from the cache. The cache is cleared automatically when an update is created, edited, solved, or rated. So the numbers stay up to date to the minute under normal traffic.

For very large stores you can change the cache window under Settings → Cache.

Export

Each section has a small CSV export button. Useful for pulling into Google Sheets for custom reporting or for an end-of-quarter team review.
