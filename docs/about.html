About the maker

One person, full review of every change

Hi, I’m Ankur Vishwakarma — a WordPress Architect based in India. I’ve spent the last 11+ years building scalable WordPress and WooCommerce platforms, and I built Order Updates for WooCommerce because I kept seeing the same gap.

Customer support lives in one tool. The order lives in another. The team copies context between them all day, important conversations get lost between systems, and customers keep asking the same questions because no one knows what was already answered.

This plugin keeps the conversation where the order is. Free, open source, no per-seat pricing, no tracking, no SaaS dependency. One person making it — full review of every change, modern WordPress coding standards throughout, 215+ unit tests on the security-critical paths.

About support: support is me, on GitHub. I read every issue, every reply on the WordPress.org forum, every email. Critical bugs get fixed fast. Feature requests get queued openly so you can see what’s next. No per-seat tier, no SaaS lock-in to abandon — the plugin keeps working even if I take a month off.
